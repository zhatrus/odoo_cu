# Курсова робота по навчальному курсу "Базовий курс для розробників"

# Тема

Тема курсової роботи обирається на розсуд студента курсу.

# Завдання

Вимоги до реалізації курсової роботи наступні:

1. **Моделі**: у модуль потрібно додати не менш ніж 5 нових моделей. 

2. **Представлення**: кожна модель повинна мати два базових представлення (*tree, form*) в обов'язковому порядку, та мати хоча б одне з наступних представлень (*pivot, graph, kanban, search*) в одній з додаваних моделей.

3. **Меню**: модуль повинен мати власне меню верхнього рівня. Необхідно додати зображення для батьківського розділу меню.

4. **Наслідування**: модуль повинен наслідувати та розширювати (або змінювати) функціонал якнайменше двох стандартних моделей.

5. **Візарди**: необхідно додати тимчасову модель до модуля.

6. **Звіти**: модуль повинен містити один друкований звіт.

7. **Демо дані**: необхідно додати демо дані до модуля, що розробляється, якнайменше по 3 записи для кожної доданої моделі. 

8. **Тести**: модуль повинен мати тести, для цього бажано продумати декілька тест-кейсів та додати їх у модуль. Кожна додана модель повинна мати хоча б по одному тестовому методу.

9. **Права доступу**: 

   1. Додайте дві нові групи користувачів (User та Admin) до модуля, котрі повинні входити в одну категорію та мати ієрархічну структуру. 

   2. У файлі ir.model.access.csv потрібно надати права на доступ до додаваних моделей для цих двох груп користувачів. 

   3. Додайте у модуль не менш ніж 2 правила доступу.

10. **Опис / документування**: 

    1. Необхідно додати у модуль файл README.rst.

    2. Додати "docstring" до всіх методів та класів.

    3. Використовуйте коментарі у коді для пояснення складної логіки.

    4. Додайте файл index.html у модуль, з використанням, щонайменш, однієї картинки.

11. **Переклади:** модуль повинен мати переклад на українську мову.

12. **Перевірка якості кода**: 

    1. Модуль повинен проходити перевірки лінтерами (pylint та flake8 з пакета скрипту odoo-helper) без зауважень.

    2. Модуль повинен встановлюватися та видалятися з системи без помилок та попереджень.

    

# Строк здачі

Кінцевий строк здачі курсової роботи: **тиждень після проведення зустрічі по останньому 6 блоку курсу**.
